# Component A: The World Builder

## 1. Core Responsibility
負責建構戰鬥發生的 **物理/數據環境 (Environment)**、**世界法則 (Laws)** 與 **敵人生態 (Enemy Ecology)**。

## 2. Environmental Logic (環境邏輯)
*   **Data Characteristics**: 定義戰場的數據屬性。
    *   *Examples*: `Low Gravity Code Zone` (低重力代碼區), `Corrupted Neon Slums` (錯誤溢出的霓虹貧民窟), `Infinite Library` (無限圖書館).
*   **The Root Law Interference**: 描述根律如何干涉環境。
    *   *Visuals*: 空氣中漂浮的二進制塵埃、牆壁上流動的發光電路、重力異常導致的碎石懸浮。

## 3. Enemy Ecology (敵人生態)
*   **Origin**: 敵人是舊世界的殘留數據 (Legacy Data)、防毒軟體具象化 (Anti-Virus Protocol)、還是被污染的使徒 (Corrupted Apostle)？
*   **Visuals**:
    *   **Damage Response**: 敵人受傷時不流血，而是逸散出 **帶透明度的賽博青資訊方塊 (Cyber Cyan Information Cubes)**。
    *   **Death**: 死亡時崩解為 **黑色線框 (Dark Wireframes)**，核心炸裂出對應屬性的鑽石塵。
