# Genesis Line: Style Guide (美術規範庫)

## 1. Color Palette (標準色票)
*   **Cyber Cyan**: `#00FFFF` (數據、靈魂、根律、友方特效)
*   **Magenta Neon**: `#FF00FF` (干擾、霓虹、錯誤、敵方警告)
*   **Deep Ink**: `#0A0A0A` (陰影、背景基底、虛無)
*   **Warning Yellow**: `#FFD700` (Ren 的雷電、系統警告)
*   **Crimson Red**: `#FF0000` (Retsu 的火焰、致命錯誤)

## 2. Texture Keywords (材質關鍵字)
*   **Glitch**: `Chromatic aberration, datamoshing, pixel sorting, screen tearing`.
*   **Magic/Tech**: `Holographic glyphs, floating kanji, wireframe geometry, optical fiber trails`.
*   **Destruction**: `Crystallized fragments, voxel debris, liquid ink splatters`.

## 3. Camera Language (鏡頭語言)
*   **Obari Pose**: 強調透視的帥氣站姿，武器指向鏡頭。
*   **Impact Frame**: 瞬間的黑白/反色閃爍，強調打擊感。
*   **Dutch Angle**: 傾斜鏡頭，營造不安與動態感。
