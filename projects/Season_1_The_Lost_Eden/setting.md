# 📖 設定集：失落的伊甸 (Season 1 Setting: The Lost Eden)

> **「起初，創世者編寫了代碼。代碼是完美的，代碼就是光。」**
> **"In the beginning was the Code, and the Code was with the Architect, and the Code was Light."**

## 1. 核心哲學 (Core Philosophy)

本季度的核心美學建立在 **「神學與科技的鏡像 (Theology in Technology)」** 之上。
在這個高度發達的賽博宇宙中，科技已經進化到極致，以至於「系統維護」看起來就像是「宗教儀式」，而「程式除錯」則是一場「神聖的驅魔」。

* **表象 (Skin)**：硬派賽博龐克、霓虹光影、機甲、數據流、程式碼。
* **骨幹 (Bone)**：舊約聖經隱喻、伊甸園、原罪、獻祭、救贖。

## 4. 核心衝突 (The Conflict)

### **律法 vs. 恩典 (System vs. Grace)**
* **樞紐的觀點**：「這個世界有 0.1% 的錯誤代碼。為了保護整體，必須刪除該世界。」（功利主義/律法主義）
* **小隊的觀點**：「哪怕只有 0.1% 的良善，這個世界就值得被救贖。」（人本主義/恩典）

這場戰鬥不是為了推翻創世者，而是為了**推翻僵化的「代理系統（律法）」**，讓人直接遇見神。

---

## 5. 視覺美學指引 (Visual Aesthetics)

* **光影**：強調 **「神聖光 (God Rays)」** 與 **「深淵黑 (Void Black)」** 的對比。樞紐場景應有大教堂般的垂直光束；被感染區域則應有如同瀝青般的黑色黏稠感。
* **特效**：
    * **紋路 (Lines)**：不要畫成單純的霓虹燈，要畫成 **「流動的文字」** 或 **「發光的血管」**。
    * **雜訊 (Static)**：不是電子故障，而是 **「有機的潰爛」**。像素像蟲一樣吞噬畫面。
* **構圖**：大量使用 **「仰角 (Low Angle)」** 來展現角色的崇高與世界的壓迫感。模仿文藝復興時期的宗教畫構圖，但角色穿著機能風 (Techwear)。
